# SantiSZR Web 版操作说明书

适用于当前项目：`D:\SantiSZR`

- 前端地址：`http://127.0.0.1:5173`
- 后端地址：`http://127.0.0.1:7860`

## 一、启动方式

运行：

```powershell
powershell -ExecutionPolicy Bypass -File D:\SantiSZR\scripts\restart-web.ps1
```

这个脚本会自动重启后端和前端。

局域网访问时，用本机 IP：

```text
http://本机IP:5173
```

例如：

```text
http://192.168.31.200:5173
```

注意：局域网访问只是远程打开控制页面，涉及本机文件、半自动发布、浏览器自动化时，实际操作仍发生在运行后端的这台电脑上。

## 二、基本流程

推荐按页面从上到下使用：

1. 工作空间
2. 文案 / 改写
3. 音频
4. 数字人
5. 字幕
6. 发布
7. 任务中心
8. 设置 / 诊断

## 三、工作空间

第一次使用时选择一个目录作为工作空间，例如：

```text
D:\test
```

后续生成的音频、视频、字幕、发布素材都会存到这个工作空间里。

常见目录：

```text
D:\test\tts
D:\test\avatar
D:\test\subtitle
D:\test\publish
D:\test\reference\audio
D:\test\reference\video
```

软件重新打开后，会自动进入上一次使用的工作空间。

## 四、文案 / 改写

这里用于准备后续生成音频的文本。

操作：

1. 输入原文、分享文案、链接或本地素材路径
2. 点击“提取原文”
3. 如需改写，先到设置里配置大模型
4. 点击“改写文案”
5. 改写后的文案会作为音频生成的主要文本来源

如果没有配置大模型，改写不会假装 AI 生效，会提示需要配置。

## 五、音频模块

用途：根据文案生成语音。

操作：

1. 上传或选择参考音频
2. 填写或使用上游文案作为生成文本
3. 可选开启“极致克隆”
4. 点击“生成音频”
5. 在音频列表中试听、选择音频

极致克隆说明：

- 默认关闭
- 关闭时走普通参考音频克隆流程
- 开启后会使用参考音频文案进行精准匹配
- 参考音频文案会自动识别，也可以手动修改

音频列表只显示当前工作空间里的音频。

## 六、数字人模块

用途：用音频 + 参考视频生成数字人视频。

操作：

1. 上传或选择参考视频
2. 确认已经选择音频
3. 点击“生成数字人”
4. 生成后右侧会显示结果视频

注意：

- 数字人视频本身不依赖字幕
- 当前设计是先生成无字幕视频
- 字幕作为后处理单独加载到视频中

## 七、字幕模块

用途：根据音频生成 SRT 字幕，并可把字幕加载到视频中。

操作：

1. 确认已选择音频
2. 点击“生成字幕”
3. 在字幕预览中检查或修改字幕文本
4. 如果需要，把字幕加载到视频中
5. 点击“加载字幕到视频”

AI 纠错说明：

- 只修正错别字、多音字、明显识别错误
- 不会润色改写
- 不会改变字幕行数
- 不会改时间轴
- 目的是保持字幕和音频对齐

## 八、发布模块

用途：准备发布素材，并半自动打开平台发布页。

发布素材包括：

```text
D:\test\publish\output.txt
D:\test\publish\output_cover_with_text.png
```

操作：

1. 确认已有成片视频
2. 点击“AI生成”
3. 系统会根据音频文案生成标题、标签、描述、封面标题和高亮词
4. 可调整抽帧时间、封面标题、高亮词
5. 封面会自动重新生成预览
6. 选择发布平台
7. 点击“打开发布页”

标签格式：

```text
#标签1 #标签2 #标签3
```

半自动发布说明：

- 会在本机 Chrome 打开抖音 / 小红书 / 视频号发布页
- 尝试自动上传视频、封面
- 尝试填入标题、描述、标签
- 最后需要用户自己检查并点击发布
- 如果页面结构变化，可能需要手动补填

局域网访问时，发布页仍然会在运行后端的本机电脑上打开。

## 九、设置 / 诊断

这里主要配置：

- API Base
- 大模型 API Key
- 大模型 API Base
- 默认模型

大模型默认可用 DeepSeek，例如：

```text
API Base: https://api.deepseek.com/v1
Model: deepseek-chat 或 deepseek-v4-flash
```

配置后建议点击“测试”，确认连接成功。

## 十、任务中心

任务中心显示后台任务状态，包括：

- 文案提取
- 改写
- 音频生成
- 数字人生成
- 字幕生成
- 字幕后处理
- 发布任务

如果点击按钮后没反应，先看任务中心和发布状态区域，通常会显示失败原因。

## 十一、常见问题

### 页面没更新

刷新浏览器。

### 前后端异常

运行：

```powershell
powershell -ExecutionPolicy Bypass -File D:\SantiSZR\scripts\restart-web.ps1
```

### AI 生成结果像文件名

检查音频文案是否为空。发布 AI 当前应使用音频文案作为输入；如果没有音频文案，AI 会缺少有效上下文。

### 半自动发布停在上传页

1. 确认平台已登录
2. 重新点击“打开发布页”
3. 如果平台改版，可能需要手动上传视频或补填内容

### 局域网访问时发布页在哪里打开

发布页会在运行后端的本机电脑上打开，不会在局域网远程访问的电脑上打开。
